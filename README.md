# quiz

#  Run frontend (Angular)
1. Checkout the project
2. cd frontend
3. npm install
4. ng serve

# Run backend
1. cd api
2. npm install
3. npm start

# Setup DB(Mysql DB)
1. Import the quiz.sql file 

# .env for api project
1. Create an .env file in api project and copy the below code in .env file
